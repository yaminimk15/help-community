# 🆘 Help Community Web App

A full-stack web application that connects people who need help with those who can provide it.  
Users can post help requests, mark urgency levels, and track request status in a clean and modern dashboard.

---

## 🚀 Features

- 🔐 User Login & Signup
- 🆘 Create Help Requests
- 🚨 Urgency Levels (Low / Medium / High)
- 📊 Status Tracking (Pending / Accepted / Completed)
- 🤝 “I Can Help” feature
- 📱 Modern UI with dashboard
- 🔍 Filters (All / My Requests / High Priority)
- 📊 Live Statistics (Total / High / Completed)
